This Python GUI application performs CDS correction and ΔA(λ), ΔE(λ), ΔS(λ) decomposition from multi-sample datasets 

It requires an Excel file with headers formatted as CIDS-<sampleID>-<concentration>, CD-<sampleID>-<concentration>, and ST-<sampleID>-<concentration> (sample IDs may contain dashes). The program groups complete triplets automatically, clamps low scattering extinction values (ST ≥ 0.01) for numerical stability, and solves for ΔA(λ), ΔE(λ), and ΔS(λ) using the corrected model (with constant path length l_T, default 0.83, used in our experiments). It then forward-predicts CDS for validation, plots ΔA, ΔE(λ), and ΔS(λ), and exports two Excel sheets: a full decomposition file and a ΔA-only file, ΔE(λ)-only file, and ΔS(λ)-only file for downstream analysis.
